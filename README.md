1. **用例图**：展示了医院管理系统中各个参与者（如医生、护士、患者等）与系统之间的交互关系，明确了系统的功能需求。

3. **顺序图**：展示了系统中各个对象之间的交互顺序，帮助理解系统的动态行为。

4. **活动图**：描述了系统中的业务流程，展示了各个活动之间的流转关系，帮助理解系统的业务逻辑。

5. **状态图**：展示了系统中对象的状态变化及其触发条件，帮助理解系统的动态行为和状态转换。
